# E-Commerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze e-commerce sales performance and identify key business insights.

## 📊 Key Features
- Total Sales, Profit, Quantity, and Average Order Value (AOV) KPIs
- Sales analysis by category, sub-category, state, and customer
- Monthly profit trend analysis
- Payment mode-wise quantity distribution
- Interactive filters and slicers

## 🛠 Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization
